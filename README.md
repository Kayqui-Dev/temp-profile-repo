## 🤼‍♂️ Sobre mim

Sou desenvolvedor **Full-Stack** focado em arquiteturas de alto rendimento, automações cognitivas com IA e integrações resilientes. Em paralelo, sou atleta competidor de **Wrestling** (Luta Olímpica) pela equipe NTG, Medalhista de Bronze no Campeonato Brasileiro Sub-18 (Série Ouro) e Vice-Campeão Paulista.

> "No wrestling não há atalhos; cada vitória é conquistada com repetição exaustiva no tatame. No código é igual: sistemas complexos e performantes nascem da disciplina do commit persistente."

<hr>

## 💼 Kodava Solutions

Fundador da **Kodava Solutions**, onde desenvolvo ecossistemas digitais inteligentes de ponta a ponta:
- **Agentes Inteligentes de IA:** Automações complexas, assistentes autônomos integrados a APIs.
- **Sistemas Web Premium:** Desenvolvimento robusto de aplicações com foco em design excepcional e performance extrema.
- **Integração de Sistemas:** Conectores customizados via n8n, Supabase, APIs RESTful e Webhooks.

<hr>

## 🚀 Especialidades
- **Desenvolvimento Web & SaaS:** Next.js, React, TypeScript, TailwindCSS, Node.js e Python.
- **Engenharia de IA & Automação:** Orquestração de LLMs, pipelines cognitivos, chatbots avançados e fluxos de trabalho no n8n.
- **Arquitetura de Dados:** Modelagem relacional com PostgreSQL, segurança no Supabase e banco de dados em tempo real.
- **Alta Performance:** Otimização de Web Vitals, performance e deploys automatizados (CI/CD).
